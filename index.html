Instructions:

Try to move the tiles around so that you can reach the goal state in the least number of moves.

In each move you can place a tile, which is adjacent to the empty space, inside the empty space.

Use arrow keys to move adjacent tiles into the empty space.

  • Up arrow key : the tile below the empty space will move upwards into it
  • Right arrow key: the tile to the left of the empty space will move rightwards into it
  • Down arrow key: the tile above the empty space will move downwards into it
  • Left arrow key: the tile to the right of the empty space will move leftwards into it
